Projects

CritChat


CritChat is a platform for art students to host and join online critique spaces, with real-time chatroom functionality. Guests can react to the presented images, as well as chat freely, while hosts can upload their own work upon creating the critique spaces. Node.js and Socket.IO make up the back-end of the application, while React.js powers the client-facing components.


      

Ciel AI


Developed in Python, Ciel AI is a Deep Convolutional Generative Adversarial Network (DCGAN) used to artificially generate images of the sky, through a dataset of photographs sourced from a research centre in Germany. Over time, the competing neural networks get more successful in producing more realistic images of the sky itself.


      

runLib


runLib is a running route library, containing an editable database of running routes categorised by distance, run type, location and description. Node.js and MongoDB all play their part in the back-end, while JavaScript, HTML and CSS hold up the front-end.


      

Nine Two


Nine Two is a web application for accessing information on football clubs within the top four leagues of the English footballing pyramid. Nine Two uses data from thesportsdb.com API, and allows the user to select a league, and a respective club to discover information about. CSS animations make Nine Two feel responsive, and the information itself is displayed in a clear, digestible manner.


      

Isolation Station

      

Isolation Station is an internet radio station founded by artist and designer Spike Wright. In this website, CSS animations create a responsive online base for the station, with a linked 3D space designed in 3DVista allowing for the ultimate user experience.

The Kitchen Inventory

      

The Kitchen Inventory is a useful web application to keep track of what is in the cupboards of the kitchen. Items can be easily added or removed from the kitchen, and a shopping list is provided where items can be deleted or transferred into the kitchen when purchased.

RentEst

      

RentEst is a tool for working out an estimate on how much rent you should be paying per month in the U.K. The program uses data from Statista on the household rent to income ratio in various areas across the United Kingdom, and has been developed using React and React Hooks.
